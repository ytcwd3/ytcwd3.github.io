### 步骤2：配置Giscus留言板（3分钟）

1. **启用Discussions**
   - 访问：https://github.com/ytcwd3/ytcwd3.github.io/settings
   - 勾选 "Discussions"

2. **安装Giscus App**
   - 访问：https://github.com/apps/giscus
   - 点击 "Install"
   - 选择你的仓库

3. **获取配置参数**
   - 访问：https://giscus.app/zh-CN
   - 输入仓库：`ytcwd3/ytcwd3.github.io`
   - 复制生成的 `data-repo-id` 和 `data-category-id`

4. **更新配置**
   - 打开 `script.js`
   - 找到 `initGiscus()` 函数
   - 替换 `YOUR_REPO_ID` 和 `YOUR_CATEGORY_ID`
   - 提交并推送

### 步骤3：测试（1分钟）

1. 等待GitHub Pages部署（1-2分钟）
2. 访问你的网站
3. 测试留言板功能
4. 检查加载速度

## 📝 Excel转JSON工具使用

### 准备Excel文件

创建一个Excel文件，包含以下sheet：

- 任天堂主机
- NS
- 任天堂掌机
- 索尼
- 其他

每个sheet包含以下列：

```
name | category | subCategory | code | unzipCode | quarkPan | baiduPan | thunderPan | updateDate
```

### 运行转换

```bash
# 安装依赖
pip install -r requirements.txt

# 运行工具
python excel_to_json_converter.py
```

或者在Windows上：

```bash
# 打包成exe
build_exe.bat

# 运行exe
dist/Excel转JSON工具.exe
```

## ❓ 常见问题

### Q1: Giscus留言板不显示？

**A**: 检查以下几点：

1. Discussions是否已启用
2. Giscus App是否已安装
3. repo-id和category-id是否正确
4. 浏览器控制台是否有错误

### Q2: Excel转JSON失败？

**A**: 确保：

1. Excel文件格式正确
2. 包含所有必需的列
3. 已安装Python依赖

### Q3: 网站加载还是慢？

**A**:

1. 确认data文件夹已上传
2. 考虑实施方案B（按需加载）
3. 使用CDN加速

### Q4: 如何回退？

**A**:

```bash
git checkout script.js.backup
git commit -m "回退到原版本"
git push
```
